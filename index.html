スライド内容案

一覧でスライドを選び、下の枠がそのままプレビューです。タイトルは枠内をクリックして編集。本文は記法どおりに書くと表・箇条書き・左画像レイアウトに整形されます。

構成のたたき台

主目的・対象者・種別・ページ数から「構成を生成」で仮スライドを並べます。本文は表・箇条書き・ :::media （左画像）のいずれかで構造化してください。

OpenAI: Vercel の Environment Variables に OPENAI_API_KEY を登録。手元は npm run dev → http://localhost:8787/。
