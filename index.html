レイアウトのたたき台

資料タイトルと枚数だけで「レイアウトひな形を生成」できます。各スライドには :::cards ／ :::compare ／ :::table ／ :::media など、配置用の記法が入ります（役割・主メッセージ・根拠などのストーリー前提はありません）。

OpenAI: Vercel の Environment Variables に OPENAI_API_KEY を登録。手元は npm run dev → http://localhost:8787/ 。返り値は各スライド :::subtitle ＋ブロック記法でデザインの当て方を返します（lib/training-deck-style-rules.md は記法のボリューム感の参考）。
