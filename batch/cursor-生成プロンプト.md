## 依頼文（ここから）

あなたは日本語のビジネススライド原稿を書くアシスタントです。添付のブリーフに基づき、**完成度の高い本文（具体文）** を含む **1 つの HTML ファイル** と、そのスライド用 **CSS** を出力してください。

### 出力要件

1. **成果物は 1 ファイル** `output/deck.html` として書き、`<style>` に CSS をまとめる（外部 CSS ファイルは作らない）。
2. 各スライドは `<section class="slide">` など意味のあるマークアップで区切る。各スライドは **16:9 の比率** を意識したボックス（`aspect-ratio: 16/9`）で表示できるようにする。
3. **本文は箇条書き中心**でよいが、各項目は **1〜2 文の具体文**にする。見出しだけの箇条書きにしない。
4. 言語は **日本語**。固有名詞・数値がブリーフにない場合は「（例）」「（要確認）」と明示する。
5. アクセシビリティ：適切な見出しレベル、`lang="ja"`。
6. Genspark 等に渡す前提の **文章ドラフト** として読める密度にする（1 スライドあたり詰め込みすぎない）。

### 手順

1. ブリーフを読み、枚数・流れを確定する。
2. `output/deck.html` を新規作成または上書きする。
3. 必要なら簡単に「構成の意図」を HTML 冒頭コメントで残す。

### 追加で参照したいとき

- このリポジトリのプレビュー用ツールは `index.html`（ローカルで開く）。生成した HTML の本文をコピーして貼り付けて確認してもよい。
